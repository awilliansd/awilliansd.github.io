# awilliansd.github.io

Site pessoal e currículo online de **Alessandro Willian de Souza Dias**.

Publicado com GitHub Pages em: [https://awilliansd.github.io](https://awilliansd.github.io)

## Sobre

Este repositório contém o código-fonte do meu site pessoal, onde apresento minha trajetória profissional, habilidades técnicas, projetos, recomendações e formas de contato.

## Tecnologias

- **HTML5** semântico
- **CSS3** com variáveis, animações e layout responsivo
- **JavaScript** vanilla
- **Font Awesome** para ícones
- **Google Fonts** (Inter + JetBrains Mono)

## Funcionalidades

- 🌙 Tema **dark mode** moderno
- 🌍 Navegação em **Português** e **Inglês**
- 📱 Layout totalmente responsivo
- ✨ Animações suaves ao rolar a página
- 🔗 Links diretos para LinkedIn, GitHub, e-mail e projetos
- 📄 Download dos CVs em PDF

## Estrutura

```
.
├── index.html          # Página principal
├── style.css           # Estilos do site
├── script.js           # Internacionalização e interatividade
├── README.md           # Este arquivo
└── *.pdf               # Currículos em português e inglês
```

## Como rodar localmente

Como o site é estático, basta abrir o arquivo `index.html` diretamente no navegador. Alternativamente, use um servidor local simples:

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .
```

Depois acesse `http://localhost:8000`.
